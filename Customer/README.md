# Digital Concierge Frontend

This folder contains the frontend structure for the Digital Concierge Ecosystem.

- Vite + React + TypeScript + Tailwind CSS
- Redux Toolkit + React Query + React Router + Axios
- Feature-first customer portal architecture
- Modular customer functionality under `src/features/customer`

## Structure

- `public/`
- `src/app/`
- `src/assets/`
- `src/shared/`
- `src/layouts/`
- `src/features/`
- `src/styles/`
- `src/tests/`
- `src/mocks/`
- `src/env/`

## Run

```bash
cd frontend
npm install
npm run dev
```
